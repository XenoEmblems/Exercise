Yleana Coding Exercise

This exercise will be distributed to potential applicants for the software developer position at Yleana.

User stories

1. As a developer, I would like a utility, triggered by a button click, which reads information with an ajax call from an SQL table called "Customers" and display it to the user in tabular format. This will let end-user see all the customers inside its database.

2. As a developer, I would like a utility, triggered by a button click, which writes information with an ajax call to an SQL table called "Customers". This will let end-user add new customers to its database.

Initial Instructions/Guidance

  1. Create an account on Github
  2. Install GitExtensions, vanilla Git for windows, or Github for Windows
  3. (Optional, not necessary if using Github for Windows) Set up your local machine to allow communicating with Github
  4. Navigate to the Yleana Github Account
  5. Select the "Yleana.Exercise" project
  6. Click the Fork button (Forking instructions)
  7. Clone the repository (Git Extensions Manual, also see Forking instructions)
  8. Code!
  9. When finished, push to your Github fork (you can do this as much as you would like to)
  10. Perform a pull request
